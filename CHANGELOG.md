# Changelog

Все заметные изменения проекта. Формат основан на здравом смысле, версии — по
истории разработки скрипта.

## v1.14.3

### Сетевой монитор
- Анти-мерцание: экран перерисовывается на месте (`SetCursorPosition`) вместо
  `Clear-Host` на каждом тике — курсор больше не прыгает при автообновлении.
- Сортировка (`S`): Состояние → Процесс → Гео → Адрес.
- Фильтр/исключение (`F`): Все → Активные → Без LISTEN → Только внешние.
- Пауза (`P`) — заморозить автообновление.
- Длинный список обрезается по высоте окна (показывается «… ещё N»).

### Меню
- Кнопка `L` теперь переключает легенду (вкл/выкл) в правой колонке, а не
  открывает отдельный overlay. В режиме «выкл» меню одноколоночное и IP/Geo
  видны полностью.

### Прочее
- `Sanitize-Domain` ужесточён: битый ввод вида `://google.com`, `:`, `-` больше
  не уходит в DNS-резолв; добавлена валидация домена.
- Очищены списки `lists/google.txt` и `lists/services.txt` (мусор и дубликаты).
- Удобный запуск: `NetworkChecker.bat` (двойной клик).

## v1.14.2

- Удалён дубль функции `Show-Legend` и мёртвый код `Show-SideHelp` (~170 строк).
- Исправлены ссылки на имя файла в Руководстве (`v1_14.ps1`).
- Документация: переписан `README.md`, добавлен `MANUAL.md`.
