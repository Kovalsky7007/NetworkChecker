# NetworkChecker — простыми словами

Эта страница — для тех, кто не разбирается в сетях. Без сложных слов. Просто
делай по шагам.

> Программа отвечает на один вопрос: **«почему у меня не открывается сайт —
> он сам лёг, или его кто-то блокирует?»**

---

## 1. Как запустить

1. Найди файл **`NetworkChecker.bat`**.
2. **Дважды кликни** по нему.
3. Выскочит окошко Windows «Разрешить внести изменения?» — нажми **«Да»**
   (программе нужны права администратора, иначе она не сможет проверять сеть).
4. Откроется чёрное окно с меню. Готово.

Если Windows ругается на «политику выполнения» — просто запускай через `.bat`,
он всё делает сам.

---

## 2. Что нажимать в меню

Просто набери цифру и нажми **Enter**.

| Кнопка | Что делает простыми словами |
| ------ | --------------------------- |
| **2** | Проверить кучу сайтов сразу. Набери `all` — проверит все. |
| **6** | Проверить **один** сайт подробно. Введи, например, `youtube.com`. |
| **4** | Проверить, **не врёт ли твой интернет про адреса сайтов** (см. ниже). |
| **1** | Показать, какие программы сейчас лезут в интернет. |
| **7** | Справка. |
| **8** | Переключить язык (русский / английский). |
| **0** | Выход. |

---

## 3. Как понять результат

После проверки каждый сайт получает **слово-вывод** в первой колонке.
**Смотри только на него**, остального можно не понимать:

| Слово | Что это значит | Цвет |
| ----- | -------------- | ---- |
| **Доступен** | Всё хорошо, сайт работает. | зелёный |
| **Без страницы** | Сайт жив, просто на этом адресе нет главной страницы. **Это нормально**, не блокировка. | голубой |
| **Блок …** | Сайт **заблокирован** (провайдером или государством). | красный |
| **Заглушка** | Тебя перенаправляют на «заглушку» вместо настоящего сайта — тоже блокировка. | красный |
| **Нет DNS-записи** | У этого адреса просто нет записи. Часто это служебные адреса, **бояться не надо**. | серый |

Внизу будет строчка-итог, например:
```
Итог: 149   Доступно:67  Без стр.:38  Блок:16  Нет DNS:28
```
Читается так: «67 сайтов работают, 16 реально заблокированы, остальное —
мелочи, на которые не надо обращать внимания».

> 💡 Если видишь **`PING:timeout`** или **`LOSS:100%`** у работающего сайта —
> это **не страшно**. Многие сайты (особенно банки) специально не отвечают на
> «пинг». Смотри на слово-вывод, а не на пинг.

## 4. «DNS-проверка» (кнопка 4) — зачем она

DNS — это «телефонная книга интернета»: превращает имя сайта (`youtube.com`)
в адрес (числа). Иногда провайдер **подменяет** эту книгу — и вместо настоящего
сайта ты попадаешь на заглушку. Кнопка **4** это ловит.

**Как пользоваться:**
1. Нажми **4**.
2. Введи сайты через запятую, например: `facebook.com,instagram.com`
3. Где спрашивает «Доп. DNS-серверы» — просто нажми **Enter** (если не знаешь).
4. Смотри на вывод внизу каждого сайта:
   - **«совпадает с DoH»** — всё честно, подмены нет.
   - **«ПЕРЕХВАТ DNS :53»** — твой интернет **подменяет адреса**, тебя обманывают.

Если увидел «ПЕРЕХВАТ» — значит сайт блокируют через подмену адресов.

---

## 5. Что делать, если сайт заблокирован

- **`Блок`/`Заглушка`/`ПЕРЕХВАТ`** — это блокировка не на твоей стороне, сама
  программа её не уберёт. Обойти можно через **VPN** или **шифрованный DNS**
  (DoH): в Windows 11 → *Параметры → Сеть и интернет → свойства подключения →
  «Назначение DNS-сервера» → включить шифрование*. В браузере — найти настройку
  «Secure DNS / Безопасный DNS».
- **`MITM`** у банка или госсайта — скорее всего **ложная тревога** от твоего
  антивируса (Kaspersky/ESET «вскрывают» трафик). Это не страшно.

---

## 6. Сохранить отчёт

После любой проверки появится строчка `S = сохранить лог`. Нажми **`S`** —
программа сохранит файл в папку **`Logs\`**. Этот файл можно отправить в
техподдержку: там уже есть всё нужное (дата, твой адрес, все результаты).
