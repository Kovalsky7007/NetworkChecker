# Документация NetworkChecker

Здесь собрана вся подробная документация проекта. Краткое описание — в

| Файл | О чём |

| [BUGS_AND_FIXES.md](BUGS_AND_FIXES.md) | Найденные проблемы и баги, причины и фиксы, как проверялось. |
| [CHANGELOG.md](CHANGELOG.md) | История изменений по версиям. |

С чего начать:

- **Пользователю** → [OVERVIEW.md](OVERVIEW.md), затем [MANUAL.md](MANUAL.md).
- **Разработчику** → [TECHNICAL.md](TECHNICAL.md) и [BUGS_AND_FIXES.md](BUGS_AND_FIXES.md).
